# Social Network API

This repository contains a simple, efficient backend for a social networking website. the routes being tested in the video are:

Users:
* Get all
* Get one by id
* Post new
* Update one by id
* Delete one by id

Friends:
* Add to user
* Remove from user

Thoughts:
* Get all
* Get one by id
* Post new
* Update one by id
* Delete one by id

Reactions:
* Add to thought
* Remove from thought

You can find the demo video [here](https://drive.google.com/file/d/1yXXs9321aRX6IwYOjFXyLqSZXThDa8Qp/view)
